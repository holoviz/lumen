# :material-earth: Earthquakes

Global earthquake data with magnitude and depth analysis.

![Earthquakes Dashboard](https://raw.githubusercontent.com/holoviz/lumen/main/doc/_static/earthquakes.png)

## Features

- **Global map** - Earthquake locations worldwide
- **Magnitude analysis** - Distribution and trends
- **Depth visualization** - Earthquake depth patterns
- **Time filtering** - Filter by date range

## Run this example

Save the YAML above as `earthquakes.yaml` and run:

```bash
lumen serve earthquakes.yaml --show
```
